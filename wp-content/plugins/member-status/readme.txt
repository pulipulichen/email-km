<a href="http://webweb.ca/site/products/member-status/" target="_blank">Member Status</a> shows a flashing availability status.

== Description ==

The plugin is useful if you have members/staff whoose status you want to be shown.
You need to paste either %%STATUS-AVAILABLE%% or %%STATUS-UNAVAILABLE%% in your posts/pages and
the status message will be inserted right there. 

== Installation ==

1. Upload member-status.zip into to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
