== Installation ==

1. Upload plugin to the '/wp-content/plugins/' directory</li>
2. Activate the plugin through the 'Plugins' menu in WordPress</li>
3. Navigate to Manage > Setting > Reply Comment to Email to configure the plugin.</li>
