Simple theme with a blue / steel background and rounded corners.

== Description ==

Nothing fancy, just a plain blue and gray theme with round corners.

== Installation ==

1. Unzip to the `/wp-content/themes/` directory
2. Activate theme through the 'Appearance' menu in WordPress
